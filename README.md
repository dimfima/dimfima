        🎬 SPLASH-ЭКРАН (ЭКРАН ВХОДА):
        ───────────────────────────────────
        • При заходе на сайт - чёрны экран с кнопкой ENTER
        • Пока не тыкнешь ENTER - музыка не играет,
          основной контент скрыт (экономим ресурсы)
        • После нажатия - экран мигом исчезает,
          музыка стартует с плавно нарастающим звуком
        
        
        🎵 МУЗЫКАЛЬНЫЙ ПЛЕЕР
        ───────────────────────────────────
        • Плейлист из 4 треков
        • При входе включается рандомный трек
        • Кнопки вперед/назад для переключения треков
        • Плавное затухание в паузе - нарастающее при старте
        • Ползунок громкости (направо)
        • Прогресс-бар трека (украшение, не кликабельный)
        • Показывает время: "1:23 / 3:45"
        • Автопереключение на следующий трек когда кончается
        
        
        💻 LOW DETAIL MODE (ДЛЯ СЛАБЫХ ПК, W.I.P):
        ───────────────────────────────────
        • Кнопка OFF/ON на splash-экран
        • При включении:
          - меньше частиц (150 - 60)
          - меньше текстовых символов
          - глитч-эффекты реже и проще
          - отключаем RGB-сдвиг и эхо символов
        
        
        🎨 ВИЗУАЛЬНЫЕ ЭФФЕКТЫ:
        ───────────────────────────────────
        • Летающие частицы на фоне
        • Текстовые частицы собираемые в слова
        • Глитч-эффекты: шум, полосы, сдвиги, RGB-разделение
        • Scanlines (полосы как на старых мониторах)
        • Кастомный указатель (красная точка)
        
        
        🔗 ССЫЛКИ:
        ───────────────────────────────────
        • Кнопка STEAM PROFILE (внизу)
